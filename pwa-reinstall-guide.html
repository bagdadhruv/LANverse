If your PWA is opening in browser mode instead of standalone mode, follow these steps to completely reinstall it properly.

1Uninstall Completely

Windows:

  • Right-click the LANverse desktop icon
  • Select "Uninstall" or "Remove"
  • Also remove from Start Menu if present

Mobile:

  • Long-press the LANverse app icon
  • Select "Uninstall" or "Remove"

2Clear Browser Data

Chrome/Edge:

  • Press Ctrl+Shift+Delete
  • Select "All time" for time range
  • Check "Cookies and site data"
  • Click "Clear data"

Firefox:

  • Press Ctrl+Shift+Delete
  • Select "Everything" for time range
  • Click "Clear Now"

3Close All Browser Tabs

Close ALL browser tabs and windows that have LANverse open.

Make sure no browser is accessing localhost:8443

4Restart Browser

Completely close and restart your browser to ensure a clean state.

5Go to Landing Page

Navigate to: https://localhost:8443

You should see the LANverse landing page with the "Download PWA" button.

6Install PWA

Chrome/Edge:

  • Look for the üì± icon in the address bar
  • Click "Install LANverse"
  • Accept all permissions when prompted

Firefox:

  • Click the ‚ãÆ menu in the top right
  • Select "Install LANverse"
  • Accept all permissions when prompted

Safari (Mobile):

  • Click the Share button
  • Select "Add to Home Screen"
  • Accept all permissions when prompted

7Open from Apps/Desktop

CRUCIAL: Open the PWA from your apps list or desktop shortcut, NOT from the browser!

The PWA should start at the auth page in standalone mode (no browser UI).

8Test Navigation

Navigate between pages using the bottom navbar. You should:

  • Stay in standalone mode (no browser UI)
  • Not be redirected back to browser
  • See the PWA mode indicator at the top
